# SafeLane

Primera versión navegable de alta fidelidad para SafeLane: web estática, responsive y mobile-first sobre orientación, sensibilización y trabajo profesional con entidades deportivas frente a violencias en el deporte.

## Archivos

- `index.html`: inicio.
- `consulta.html`: experiencia guiada con barra de progreso y resultado orientativo.
- `como-actuar.html`: guía práctica de actuación.
- `aprende.html`: sección editorial.
- `recursos.html`: directorio de recursos.
- `entidades.html`: página B2B para captación de entidades deportivas.
- `sobre.html`: información del proyecto.
- `privacidad.html`: límites, privacidad y uso responsable.
- `styles.css`: sistema visual compartido.
- `script.js`: menú móvil y lógica de consulta.

## Publicación

La web es estática y puede publicarse gratis en GitHub Pages, Netlify o Vercel.
